## Rules

1. **No hardcoded colors** â€” only `theme-*`, `text-primary`, `var(--*)`, `glass*`, `btn-primary`
2. **No hardcoded text** â€” all strings via `t("key")` from `useLanguage()`
3. **Hydration safe** â€” providers render null/skeleton until localStorage is read
4. **Responsive** â€” grids stack on mobile (`min-[400px]:grid-cols-*`), popovers go `fixed` on mobile
5. **Animations** â€” all use CSS classes from globals.css, all respect `prefers-reduced-motion`
6. **Accessibility** â€” `cursor-pointer` on interactives, `aria-label` on icon buttons, labels on inputs
