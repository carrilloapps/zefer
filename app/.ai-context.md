# app/ — Next.js App Directory

File-system based routing. Each `page.tsx` is a thin server component that imports a `"use client"` content component.

## Pages

| File | Route | SSR? | Description |
|---|---|---|---|
| `page.tsx` | `/` | Static | Imports `HomeContent` |
| `how/page.tsx` | `/how` | Static | Imports `HowContent`, has page-specific metadata |
| `privacy/page.tsx` | `/privacy` | Static | Imports `PrivacyContent` |
| `terms/page.tsx` | `/terms` | Static | Imports `TermsContent` |
| `project/page.tsx` | `/project` | Static | Imports `ProjectContent` |
| `device/page.tsx` | `/device` | Static | Imports `DeviceContent` |

## Root Files

| File | Purpose |
|---|---|
| `layout.tsx` | Root layout: fonts, metadata, viewport, ThemeProvider → LanguageProvider → LegalBanner |
| `globals.css` | Complete design system: theme variables (dark/light), liquid glass, animations, responsive, skeleton |
| `icon.svg` | Favicon (transparent shield + checkmark) |
| `apple-icon.tsx` | Generated 180x180 Apple touch icon |
| `opengraph-image.tsx` | Generated 1200x630 OG image |
| `twitter-image.tsx` | Generated 1200x630 Twitter card |
| `manifest.ts` | PWA web manifest |
| `robots.ts` | robots.txt (allow all) |
| `sitemap.ts` | sitemap.xml (6 pages) |

## Provider Nesting Order

```
<html data-theme="dark">
  <body>
    <ThemeProvider>           ← reads theme from localStorage, shows Skeleton until ready
      <LanguageProvider>      ← reads locale from localStorage, renders null until ready
        <div>{children}</div>
        <LegalBanner />       ← consent banner, persisted in localStorage
      </LanguageProvider>
    </ThemeProvider>
  </body>
</html>
```

## globals.css Structure

1. Theme variables (dark + light) — 50+ CSS custom properties each
2. Tailwind @theme inline integration
3. Liquid glass system (glass, glass-nav, glass-banner, glass-hover)
4. Ambient gradient blobs (3 animated blobs)
5. Glow effects (glow-green, glow-green-sm, glow-pulse)
6. Key generator popover styles
7. Form elements (inputs, selects, textareas)
8. Responsive rules (safe-area-inset, iOS zoom fix, touch targets, 100dvh)
9. Focus & accessibility
10. Skeleton shimmer animation
11. 15+ animation classes (entrances, loading, reveal, error, success, etc.)
12. Theme utility classes (theme-heading through theme-step-number)
